
# Amazon Scraper




## Run Locally

Clone the project

```bash
  git clone https://github.com/Vibhav0/web-scraping.git
```

Go to the project directory

```bash
  cd web-scraping
  cd Amazon_Scrapper
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start scraping

```bash
  scrapy crawl amazon-bot
```
